# stream_monitoring
A lightweight, browser-based multi-stream video monitoring dashboard. It allows you to monitor multiple video streams simultaneously, customize on-screen labels, and receive real-time visual alerts for stream anomalies.

- **Multi-Stream Management**: Support for up to 56 concurrent video players.
- **Global Controls**: AutoPlay, Unmute All, Reload All, Pause All, and Full Screen modes.
- **Data Management**: Save and Load stream configurations via CSV files.
- **Customizable Overlays**: Adjust font size, padding, border radius, colors, and opacity for stream labels.
- **Real-Time Alert System**: Configurable thresholds for:
  - 🖤 Black Screen Detection
  - 🧊 Freeze Frame Detection
  - 🔇 No Audio Detection (Not working)
  - 📡 No Signal Detection
- **Alert Logging**: Dedicated log panel with export and clear functionality.
- **Playback Speed Control**: 0.5×, 1×, 1.5×, 2× speed adjustments.
